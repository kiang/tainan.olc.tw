花蓮光復救災資訊地圖使用教學
地圖功能說明：
  • 行政區域：點擊村里區域可查看行政資訊並填寫救災表單
  • 救災標記：地圖上的彩色標記代表不同類型的救災資訊
  • 村里名稱：放大地圖至13倍以上可看到村里名稱標籤
救災標記圖例：
  • 🙋 需要志工 - 志工支援需求
  • 📦 需要物資 - 物資補給需求
  • 🚿 提供洗澡點 - 盥洗設施
  • 🏠 提供住宿點 - 臨時住宿
  • ❓ 其他 - 其他救災資訊
操作說明：
  • 定位：點擊「定位」按鈕找到您的當前位置
  • 座標跳轉：點擊「座標」按鈕輸入經緯度跳轉到特定位置
  • 搜尋過濾：在搜尋框輸入關鍵字過濾標記
  • 填寫表單：點擊村里區域後可填寫救災資訊表單
提醒：此地圖即時顯示透過表單提交的救災資訊，協助災區資源協調與需求媒合。
