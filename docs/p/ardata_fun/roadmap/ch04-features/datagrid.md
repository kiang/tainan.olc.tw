# 資料表格設計規格

---

## 主排行榜表格欄位

| 欄位 | 資料來源 | 排序 | 說明 |
|------|---------|------|------|
| 排名 | 計算 | 固定 | 依總金額降序 |
| 企業名稱 | （需從個別 CSV 取得） | 文字排序 | 點擊可展開詳細 |
| 統一編號 | tax_id | 無 | 8碼，等寬字體顯示 |
| 歷次捐款總額 | total_amount | 數值排序 | 預設降序 |
| 操作 | — | — | 展開/收合、回報按鈕 |

預設排序：總金額降序（最大捐款者在最前面）。

---

## 視覺規則

| 金額 | 視覺樣式 |
|------|---------|
| > 100萬 | 紅色粗體 + 脈衝 glow 動畫 |
| > 50萬 | 橘色粗體 |
| 其他 | 預設黑色 |

---

## 展開詳細（企業捐款歷史）

點擊企業列後展開子表格，顯示 `report/incomes/business/{taxID}.csv` 的內容：

| 欄位 | 說明 |
|------|------|
| 選舉 | 例：「105年立法委員選舉」 |
| 捐贈對象 | 候選人姓名 |
| 捐贈日期 | 轉換後顯示西元年月日 |
| 捐贈金額 | 格式化為台幣（加千位符） |
| 回報 | 「回報關聯案件」按鈕 |

---

## 分頁

- 每頁 50 筆
- 顯示：「第 X 頁 / 共 Y 頁」、「顯示第 A-B 筆，共 C 筆」
- 翻頁後自動捲動回表格頂部

---

## 行動裝置

表格外層使用 `overflow-x: auto`，允許水平捲動，不破版。在小螢幕上可隱藏次要欄位（如統一編號）。
