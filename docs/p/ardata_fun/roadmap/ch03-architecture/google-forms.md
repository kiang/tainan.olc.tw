# Google Forms 整合指南

---

## 為什麼用 Google Forms？

- **零成本**：無需後端伺服器，無需資料庫
- **零維護**：Google 負責表單運作、資料儲存、備份
- **可匯出**：所有回報自動進入 Google Sheets，可下載為 CSV
- **匿名友善**：不需要帳號即可提交

---

## 建立表單步驟

### Step 1：新增 Google 表單

1. 前往 https://forms.google.com/
2. 點擊「空白表單」
3. 標題設為：**「政治獻金關聯案件回報」**
4. 說明文字：
   ```
   感謝您參與公民調查！
   您的回報屬個人初步合理懷疑，不代表法律認定。
   請填入可查證的公開連結（採購網、新聞等）。
   ```

### Step 2：建立欄位

**Section 1：系統自動帶入（短答案類型）**

| # | 欄位名稱 | 必填 |
|---|---------|------|
| 1 | 捐款者名稱與統編 | 是 |
| 2 | 捐款日期與金額 | 是 |
| 3 | 收受候選人資訊 | 是 |

**Section 2：民眾手動填寫**

| # | 欄位名稱 | 類型 | 必填 |
|---|---------|------|------|
| 4 | 關聯標案/都計/政策名稱 | 短答案 | 是 |
| 5 | 案件發生或決標日期 | 短答案 | 否 |
| 6 | 公開資訊連結 | 段落 | 否 |
| 7 | 說明對價關係的理由 | 段落 | 是 |
| 8 | 聯絡方式（選填） | 短答案 | 否 |

### Step 3：設定

- 回覆 → 不收集電子郵件地址
- 確認訊息：「感謝您的回報！您的線索將有助於公民監督。」

## 取得 Entry ID

Entry ID 是預填 URL 中識別各欄位的參數（格式：`entry.數字`）。

**方法一：取得預填連結**

1. 表單右上角「⋮」→「取得預填連結」
2. 每個欄位填入任意測試值
3. 點擊「取得連結」，複製產生的 URL
4. URL 中的 `entry.xxxxxxxxx=test值` 即為各欄位的 ID

**方法二：檢視原始碼**

1. 開啟表單填寫頁（非編輯頁）
2. 右鍵 → 「檢視頁面原始碼」
3. 搜尋 `entry.`，找到對應的 input name 屬性

## 推薦欄位對應表

在 `index.html` 中設定：

```javascript
const FORM_BASE_URL = "https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?usp=pp_url";
const ENTRY = {
  donor:     "entry.111111111",  // 捐款者名稱與統編
  details:   "entry.222222222",  // 捐款日期與金額
  recipient: "entry.333333333",  // 收受候選人資訊
};
```

## 隱私保護

| 設定 | 說明 |
|------|------|
| 不收集電子郵件 | 保護回報者身份 |
| 不要求登入 | 確保任何人可提交 |
| 試算表不公開 | 僅管理者可存取回報內容 |
| 無追蹤代碼 | 系統不安裝 GA 或其他追蹤 |

---

## Google Sheets 管理

表單建立後，前往「回覆」→ 點擊試算表圖示 → 建立新試算表。

建議在試算表加入管理欄位：

| 欄位 | 說明 |
|------|------|
| 審核狀態 | 未審核 / 有效 / 無效 |
| 審核人 | 負責審核的人員 |
| 備注 | 補充說明 |

設定「工具 → 通知規則 → 每次表單提交時寄信」，確保新回報不遺漏。
