麻豆中央市場攤商新址地圖

找回熟悉的味道，在新的地方繼續

一座市場的七十年

麻豆中央市場（正式名稱「麻豆市四公有零售市場」）建於民國39年（1950年），曾設有71間店鋪、超過百個攤位，是麻豆最熱鬧的商業中心。從清晨的魚肉叫賣到午後的碗粿飄香，幾代人的日常生活都繞著這座市場展開。百年老店龍泉冰店也在此營業，後來更成為偶像劇《想見你》中「莫奶奶冰店」的拍攝場景。

為什麼要關？

市場建築歷經七十餘年，鋼筋裸露、結構老化，存在嚴重公安疑慮。更複雜的是，市場基地內有 77.59% 是私有土地，台南市政府數十年來未辦理徵購，僅以租用或代繳地價稅方式使用，地主、市府、攤商三方糾紛不斷。2024年5月，監察院正式糾正台南市政府，指出市府公告停止使用後卻默許攤商繼續營業、未將私有土地返還地主等多項違失。

十二年拉鋸

  • 1950年 麻豆中央市場建成啟用
  • 2014年4月 台南市政府公告停止使用，但攤商持續營業，市場進入「廢而不用」狀態
  • 2018年1月 市府規劃強制搬遷，攤商自救會約200人到場抗議。會長郭清竹帶領抗爭，向高等行政法院提起暫緩拆遷訴訟
  • 2024年5月 監察院通過調查報告，糾正台南市政府處理方式
  • 2025年 市府向經濟部爭取1,300萬元補助，完成麻豆市五市場改善工程作為安置場域。8月法院核發執行命令，要求攤商於10月底前騰空
  • 2026年3月 法院強制斷水斷電執行完成，市場處受理53件攤商搬遷申請
  • 2026年4月 攤商陸續搬遷中，本地圖持續更新新址資訊

這個地圖的用途

市場關了，但攤商們還在。他們散落到麻豆各處繼續營業——有的搬進市五市場，有的在中正路、平等路、光復路一帶找到新店面，有的就搬到原址對面。這張地圖記錄了我們找到的每一家攤商的新地址，希望老顧客能循著地圖，找回那些熟悉的味道。

如果你知道某家攤商的新地址，歡迎透過 GitHub 或 Facebook 告訴我們。
