# 資料來源規格

---

## 2.1 主要資料來源

### 監察院廉政署政治獻金公開查閱平台

| 屬性 | 值 |
|------|-----|
| **官方網址** | https://ardata.cy.gov.tw |
| **主管機關** | 監察院廉政署（非內政部） |
| **資料格式** | ZIP 壓縮檔，內含 CSV |
| **授權** | 政府資料開放授權條款 1.0（可商業使用、可修改） |
| **資料涵蓋** | 103年（2014）～114年（2025），涵蓋總統、立委、地方各類選舉 |

---

## 2.2 API 端點

### 個人（候選人）搜尋

```
GET https://ardata.cy.gov.tw/api/v1/search/individuals?page={n}&pageSize=1000
```

每筆結果包含 `downloadZip` 欄位，指向該候選人的 ZIP 下載 URL。

### 政黨搜尋

```
GET https://ardata.cy.gov.tw/api/v1/search/parties?page={n}&pageSize=1000
```

同樣每筆結果包含 `downloadZip` 欄位。

## 2.3 ZIP 內容

每個 ZIP 解壓後包含：

- `incomes.csv` — 收入明細（捐贈收入）
- `expenditures.csv` — 支出明細

`incomes.csv` 的欄位（依索引位置）：

| 索引 | 欄位名稱 | 說明 |
|------|---------|------|
| [0] | 序號 | 流水號 |
| [1] | 擬參選人/政黨 | 收受捐款的候選人或政黨名稱 |
| [2] | 選舉名稱 | 例：「107年縣(市)長選舉」 |
| [3] | 申報序號/年度 | 申報期別 |
| [4] | 交易日期 | 民國格式，有兩種：`1070921` 或 `104/09/16` |
| [5] | 收支科目 | 例：「營利事業捐贈收入」 |
| [6] | 捐贈者/支出對象 | 公司名稱或個人姓名 |
| [7] | 身分證/統一編號 | 法人為 8 碼統編，個人為身分證字號 |
| [8] | 收入金額 | 數字字串，帶 `.00` 後綴，例：`50000.00` |
| [9] | 支出金額 | 收入列此欄為空 |
| [10] | 支出用途 | 收入列此欄為空 |
| [11] | 金錢類 | 例：「支票」、「現金」 |

## 2.4 GitHub Pages 輸出（前端直接 fetch）

PHP Pipeline 處理完成後部署至：`https://kiang.github.io/ardata.cy.gov.tw/`

| 檔案路徑 | 說明 |
|---------|------|
| `report/incomes/business.csv` | 企業捐款排行榜，無標頭，格式：`total_amount,tax_id` |
| `report/incomes/business/{統一編號}.csv` | 個別企業歷次捐款明細，有標頭 |
| `report/incomes/individual.csv` | 個人捐款排行榜，無標頭，格式：`total_amount,id_prefix_name` |
| `report/incomes/individual/{前綴姓名}.csv` | 個別捐款人歷次捐款明細 |
| `report/2024_incomes_sort.csv` | 113年選舉候選人收入排行，有標頭 |
| `report/2022_incomes_sort.csv` | 111年選舉候選人收入排行 |
| `report/2020_incomes_sort.csv` | 109年選舉候選人收入排行 |

### business/{統一編號}.csv 標頭

```
選舉,捐贈對象,捐贈人,捐贈日期,捐贈金額
```

範例行：
```
107年縣(市)長選舉,林右昌,大中福國企業有限公司,1070921,50000.00
105年立法委員選舉,丁守中,燿華電子股份有限公司,104/09/16,500000
```

### business.csv 格式（無標頭）

```
17250000,05637971
5000000,12345678
```

第一欄為歷次捐款總金額，第二欄為統一編號。

## 2.5 解析注意事項

- **ROC 日期格式**：欄位 [4] 有兩種格式並存
  - `1070921` → 民國 107 年 09 月 21 日 → `new Date(107+1911, 9-1, 21)`
  - `104/09/16` → 民國 104 年 09 月 16 日 → 切割後同樣加 1911
- **金額解析**：`parseFloat('50000.00')` 即可，無需額外處理
- **編碼**：所有 CSV 均為 UTF-8
