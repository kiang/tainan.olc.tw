📋 我的陳情紀錄

🔍 關於本工具

本工具是 helpyou.tpp.org.tw 的替代介面，提供更友善的操作體驗：

  • 📝 自動儲存草稿 — 填寫過程自動儲存，避免遺失
  • 📋 案件追蹤儀表板 — 集中管理所有陳情紀錄
  • 🔗 追蹤連結管理 — 儲存官方回覆連結
  • 💾 本地儲存 — 所有資料存在您的瀏覽器中，不上傳伺服器
📌 陳情流程
  1. 在「提出陳情」填寫個人資料與陳情內容
  2. 點擊「送出陳情申請」
  3. 系統自動將資料 POST 至官方網站並在新分頁開啟，所有欄位已預填完成
  4. 在官方網站確認資料無誤，完成驗證碼後送出
  5. 取得官方受理編號後，回到「我的案件」點擊「編輯追蹤」更新資訊
  6. 定期回來更新案件狀態
🔒 隱私說明
所有資料僅儲存在您的瀏覽器 localStorage 中，本工具不會將您的個人資料傳送到任何第三方伺服器。清除瀏覽器資料時，儲存的陳情紀錄也會一同清除。
💡 陳情技巧
  • 陳情內容越具體越好，包含時間、地點、涉及人員
  • 附上照片或文件可加速處理
  • 訴求明確，避免要求過多
  • 保持禮貌，有助於獲得正面回應
