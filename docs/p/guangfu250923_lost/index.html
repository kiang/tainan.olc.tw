        花蓮光復災損與失物招領地圖使用說明
        地圖功能說明：
        • 災損/失物標記：地圖上的標記代表災損照片或遺失/拾獲物品的位置
        • 分類顯示：可切換查看災損照片、遺失物品、拾獲物品
        • 詳細資訊：點擊標記可查看詳情與聯絡方式
        資訊分類：
        • 📷 災損照片：民眾通報的災損現況照片
        • 🔍 尋找失物：民眾通報的遺失物品資訊
        • 📦 拾獲物品：民眾通報的拾獲物品資訊
        標記圖例：
        • 📷 災損照片 - 民眾災損情況通報
        • 🔴 遺失物品 - 民眾遺失物品通報
        • 🟢 拾獲物品 - 民眾拾獲物品通報
        操作說明：
        • 定位：點擊「定位」按鈕找到您的當前位置
        • 座標跳轉：點擊「座標」按鈕輸入經緯度跳轉到特定位置
        • 分類切換：點擊上方分頁標籤切換災損照片、遺失物品或拾獲物品
        • 搜尋過濾：在搜尋框輸入關鍵字過濾標記資訊
        • 通報上傳：點擊地圖上的行政區塊，在彈出視窗按「上傳災損照片或失物資訊」填寫表單
        • 清單導航：點擊右側列表中的任一項目可快速定位到地圖上對應標記
        提醒：此地圖提供災區災損或失物招領資訊媒合服務，協助民眾尋找遺失物品或通報拾獲物品。請如實填寫物品資訊，並保持聯絡方式暢通。
