🔍 關於本工具

本工具提供比 cmsweb.tainan.gov.tw 更友善的操作體驗， 所有個人資料與案件紀錄均儲存於您的瀏覽器本機，本工具不會上傳任何資料至第三方伺服器。

📝 填報功能
1
自動儲存草稿：填寫中的所有欄位自動存入瀏覽器，關閉後下次開啟自動帶入，不怕填到一半遺失。可點「儲存草稿」手動存檔，或「新增草稿」清空重填。
2
地圖定位：直接在地圖上點選事件地點，自動填入經緯度；也可點「定位到我的位置」取得目前所在座標。
3
送出案件：按下「送出」後，系統將資料直接 POST 至官方 API。送出成功後會取得受理編號，自動儲存至「我的案件」。
4
確認信件：官方系統受理後會寄發確認信至您的電子信箱，請於 7 日內點選確認連結，案件才正式受理。
📋 我的案件
1
案件清單：所有已送出的案件依狀態分類顯示（待確認／處理中／已完成／未受理）。點擊任一案件卡片可進入詳細頁面。
2
草稿顯示：若有未送出的草稿，會顯示在清單最上方，可選擇「繼續填寫」或「清除草稿」。
3
查詢官方進度：對有受理編號的案件點「查詢進度」，輸入驗證碼即可從官方 API 取得最新 ProcessStatus，結果會自動儲存供日後檢視。
4
匯入既有案件：點「匯入案件」，輸入電子信箱、受理編號與驗證碼，即可從官方系統取回案件資料並加入追蹤清單。若受理編號已存在則自動更新資料。
🗂️ 案件詳情
1
完整資訊：顯示案件基本資料、完整反映內容、事件位置，以及官方每個處理階段的承辦人、機關、回覆內容。
2
地圖顯示：若案件有經緯度，詳情頁會自動顯示國土測繪地圖並標記事件位置。
3
網址分享：每個案件詳情頁有獨立網址（以 # 案件編號為 hash），可直接分享或書籤，開啟後直接進入該案件頁面。
⚠️ 注意事項
  • 即時派工服務（需即時派員到場）：請直接撥打市政府聯合服務中心 (06) 632-9123
  • 天災或重大災害（颱風、地震等）：請於案件類別選擇「其他」，由專人派案
  • 交通違規檢舉（實名制）：請登錄 警察局首長交通檢舉信箱
  • 所有個人資料僅儲存於您的瀏覽器本機，本工具不會上傳任何資料至第三方伺服器
